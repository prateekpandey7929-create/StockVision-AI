About StockVision AI

Learn about our technical architecture, predictive indicators, and scikit-learn fitting mechanics.

Platform Overview

StockVision AI is a production-ready web platform that aggregates financial statistics using public Yahoo Finance APIs and generates closing price estimates using standard supervised machine learning.

Many financial research portals offer charts, but evaluating algorithmic predictions requires structured testing. StockVision AI addresses this by providing on-demand model training, real-time error indicators (R², MAE, RMSE), and direct comparisons against simple linear baselines.

Machine Learning Pipeline

Random Forest Regressor

Our primary forecast engine. This ensemble learning model fits 150 decision trees on random sub-samples (bagging) and averages predictions to control overfitting. Features are randomly split at each node, creating a robust predictor for complex historical patterns.

Linear Regression Baseline

A secondary model that serves as our control benchmark. It solves a ordinary least squares linear equation to map inputs to target values. If the Random Forest $R^2$ score is not higher than the Linear Regression model, it suggests the stock is following a simple linear trend.

Technical Feature Engineering

To predict the next day's close ($Close_{t+1}$), we engineer several input features computed from historical trading session variables ($Open_t$, $High_t$, $Low_t$, $Close_t$, $Volume_t$):

  • Lags 1, 2, and 3: Includes prior session closing prices to capture short-term momentum.
  • Moving Averages (SMA 5, 10, 20): Smoothes out short-term fluctuations to highlight trend support levels.
  • HL Spread: The difference between high and low prices ($High - Low$) tracks daily volatility.
  • Rolling Volatility: Standard deviation of daily returns computed over a 5-day sliding window.
